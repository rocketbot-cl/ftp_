# FTP
  
Connect and manage an FTP

*Read this in other languages: [English](README.md), [Portugues](README.pr.md), [Espa√±ol](README.es.md).*

## How to install this module
  
__Download__ and __install__ the content in 'modules' folder in Rocketbot path  



## Overview


1. Connect to FTP  
Connect to FTP to manage it with Rocketbot

2. List directory  
Lists the FTP directory in a Rocketbot variable

3. Create directory  
Create directory in the current path

4. Go to directory  
Go to the specified directory

5. Upload file  
Upload file to current directory

6. Download file  
Download the selected file

7. Delete file  
Delete file in current directory

8. Close Connection  
Close FTP Connection
